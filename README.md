# Jogo da Velha em Assembly MIPS

Este é um projeto de **Jogo da Velha (Tic-Tac-Toe)** desenvolvido em **Assembly MIPS**, utilizando o **emulador MARS** para testes e execução.

O objetivo do projeto foi colocar em prática conceitos de programação em baixo nível e manipulação de registradores, memória e lógica de controle de fluxo, aprendidos na disciplina de **Infraestrutura de Hardware**.

## Detalhes do Projeto

- **Linguagem:** Assembly MIPS
- **Ambiente:** MARS (MIPS Assembler and Runtime Simulator)
- **Disciplina:** Infraestrutura de Hardware
- **Turma:** CC-A

## Como Executar

1. Abra o **MARS**.
2. Carregue o arquivo `.asm` do projeto.
3. Compile e execute o programa.
4. Siga as instruções na tela para jogar contra a CPU.

## Observações

O jogo permite que **um jogador humano jogue contra a CPU**, marcando **X** ou **O** no tabuleiro, até que haja um vencedor ou empate.
